Section 3: Training Machine Learning Models

第 3 部分:训练机器学习模型

在这本书的第三部分，读者将了解到关于在 Azure 上训练和优化传统机器学习模型以及深度学习模型的所有信息。读者将使用 Azure 机器学习服务的能力，在 Azure 上实现和训练不同的深度神经网络 ( DNNs )。

本节包括以下章节:

  • 第六章，使用 Azure 机器学习构建 ML 模型
  • 第七章，在 Azure 上训练深度神经网络
  • 第 8 章、超参数调整和自动化机器学习
  • 第九章，Azure ML 集群上的分布式机器学习
  • 第十章，在 Azure 中构建推荐引擎
